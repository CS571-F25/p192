Tian's Grab & Go - Frontend Food Ordering Website

Developed an interactive multi-page restaurant ordering interface with user
authentication, menu browsing, dynamic filtering, paginated display, cart and checkout
workflows. Implemented an admin dashboard for food management, such as add, edit, and delete foods. Deployed publicly
using GitHub Pages.

1. This Website is accessible at: https://cs571-f25.github.io/p192/

2. After accessing the website, if it asks you to enter "Badger ID", you may "Dismiss" to continue.

3. You will need to register an account to continue, once you registered, you are a user, which only have access to the user's dashboard, in that, you can perform menu browsing, dynamic filtering, paginated display, cart and checkout
workflows. Note that the payment feature is not implemented.

4. Here is the link to a video demo of both user dashboard and admin dashboard: https://mediaspace.wisc.edu/media/Kaltura+Capture+recording+-+December+5th+2025%2C+8%3A14%3A08+pm/1_6b9wg1kl
